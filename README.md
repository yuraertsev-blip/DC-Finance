# Даймонд Канвас - Учет финансов

Mobile-first Single Page Application (SPA) для учета финансов компании «Даймонд Канвас».

Приложение синхронизирует данные через **Firebase Firestore** и работает как PWA (Progressive Web App). Разработано с фокусом на минимализм, премиальный дизайн и высокую скорость работы.

## Особенности

- **4 источника дохода:** СоцСети, Wildberries, Ozon, Яндекс — с автоматическим подсчётом итога за день.
- **Ввод данных:** Интерактивный календарь и удобная таблица для расходов (до 15 строк в день).
- **Категории расходов:** Настраиваемый справочник с наименованиями внутри каждой категории.
- **Аналитика:** Анализ доходов и расходов за выбранный период, столбчатая диаграмма, подробный отчёт по категориям.
- **Excel-отчёт:** Выгрузка с тремя таблицами на каждом листе (доходы по источникам, расходы по категориям, детальный журнал транзакций).
- **Облачная синхронизация:** Данные хранятся в Firebase Firestore и доступны с любого устройства.
- **Резервное копирование:** Экспорт/импорт данных в формате JSON.

## Технологии

- Ванильный JavaScript (ESM)
- HTML5 / CSS3 (кастомный дизайн без фреймворков)
- Firebase Firestore (облачное хранение данных)
- ExcelJS (генерация отчётов)
- Шрифты: Google Fonts (Outfit)
- PWA (Service Worker + Web Manifest)

## Установка и запуск

Проект не требует сборки (Vite/Webpack). Достаточно любого веб-сервера.

### Быстрый старт (Python)
```bash
cd DiamondCanvas_Finance
python3 -m http.server 8000
```
Откройте `http://localhost:8000`

### VS Code (Live Server)
Откройте папку проекта и нажмите **"Go Live"** (расширение Live Server).

## Конфигурация Firebase

Приложение использует проект `diamondcanvasfinance` в Firebase. Данные хранятся в коллекции `dc_finance`, документ `main_state`.

Для подключения к другому проекту Firebase замените объект `firebaseConfig` в файле `app.js`.

## Безопасность

- Все числовые данные валидируются и приводятся к целым числам перед сохранением.
- Данные синхронизируются через Firebase с автоматической обработкой ошибок.
- Экранируются нечисловые символы при вводе сумм.
